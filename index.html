Welcome to my website!
In it, I would like to use it as a portfolio to share my skills and projects I've been working into, please check the list below for details:

  • GeoPicSorter: a free and open source tool for desktop systems that's written in Java, to help users organize pictures based on geolocation groups such as streets, cities, etc. JAR is available to try it right from the box.
  • Electro Urn API: a highly simple voting API written using Spring for educational purposes, the user sends HTTP requests to it to make the interactions. Instructions on how to set it up and use it are included.
  • Simple Hashing Tool: just a GUI based tool you can use to hash text into a few different algorithms using Google's Guava library.
  • Compraventa Vehicular: an example website using PHP, for educational purposes. It's available only in Spanish.
  • Simple Jobs Board: a web board for educational purposes; companies can post jobs and manage candidates, and people can apply for jobs and manage the profile they share with the company. Roles don't work, so it's incomplete.
