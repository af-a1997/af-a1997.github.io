Software

This is a list of programs I'm working on.

  • GeoPicSorter: a free and open source tool in its early development stages, designed for desktop systems and written in Java, using Apache NetBeans; it allows users to automatically organize pictures based on geolocation groups such as streets, cities, etc. Executable JAR file is available to try it right from the box.
Published homework

The following is some of the homework I've done during my studies I thought I'd share, in hopes it might help anyone with programming and website designing, these projects are shared purely for educational purposes, meaning they're ill-suited to be tools you can use on a daily basis:

  • Electro Urn API: a highly simple voting API written using the Spring Framework, the user sends HTTP requests to it to make the interactions. Instructions on how to set it up and use it are included.
  • Simple Hashing Tool: just a really simple GUI based tool you can use to hash text into a few different algorithms using Google's Guava library.
  • Compraventa Vehicular: an example vehicle shop website using PHP and a Bootstrap template. Currently, it's available only in Spanish.
  • Simple Jobs Board: a web board for educational purposes written using the Spring Framework, a template in Bootstrap and Thymeleaf; companies can post jobs and manage candidates, and people can apply for jobs and manage the profile they share with the company.
